# Proposal: C-06-estructura-academica

## Why
The system needs to model the academic structure (Carreras, Cohortes, Materias) to manage academic operations and assignments.

## What Changes
- Implement Carrera model.
- Implement Cohorte model.
- Implement Materia model.
- Define relationships between these entities.

## Capabilities

### New Capabilities
- `academic-structure`: Carrera, Cohorte, Materia models.

### Modified Capabilities
- None

## Impact
- Core foundation for academic management.
- Enables academic operations and assignments.
