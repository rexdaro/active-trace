## ADDED Requirements

### Requirement: CRUD Carrera
WHEN un ADMIN solicita crear una carrera,
el sistema SHALL crear una `Carrera` válida (nombre, código único).

### Requirement: CRUD Cohorte
WHEN un ADMIN solicita crear una cohorte para una carrera,
el sistema SHALL crear una `Cohorte` válida asociada a dicha carrera.

### Requirement: CRUD Materia
WHEN un ADMIN solicita crear una materia,
el sistema SHALL crear una `Materia` válida (nombre, código único).
