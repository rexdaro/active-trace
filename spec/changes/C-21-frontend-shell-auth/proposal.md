# Proposal: C-21 Frontend Shell & Auth

## Why
Necesitamos un shell de frontend (SPA) funcional que maneje la autenticación y brinde una base sólida para el desarrollo de las vistas del sistema.

## What Changes
- Scaffolding React + Vite + TypeScript.
- `src/services/api.ts` (Axios client con interceptores de auth).
- `src/pages/Login.tsx`.
- Router configuración básica.

## Impact
- Nuevo directorio `frontend/` (o estructura cliente).
- Dependencias de frontend (npm/yarn).
- Nueva aplicación cliente.
